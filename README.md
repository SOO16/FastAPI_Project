# 나의 FastAPI 프로젝트
- FastAPI 써보기

## Setting
- `python -m venv .\.venv`  
- `.\.venv\Scripts\activate` 
- `pip install -r .\requirements.txt`

## 실행법

- 가상환경 만들고, `pip install -r ./requirements.txt`
- `uvicorn main:app --reload`

## API 설명페이지 접속법
- `http://localhost:8000/docs`
- `http://localhost:8000/redoc`
